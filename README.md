# Employee Data Cleaning Project (SQL)

This project demonstrates SQL data cleaning techniques using MySQL. 
The dataset included messy values such as inconsistent capitalization, 
mixed date formats, 'nan' values, and numeric text.

## Key Cleaning Operations:

- Standardized Name formatting
- Cleaned Gender values (Male/Female)
- Converted Age to integer
- Converted Salary to numeric
- Cleaned multiple date formats into YYYY-MM-DD
- Trimmed and validated Email
- Cleaned phone numbers and removed symbols
- Removal of Duplicates

## Tools used:
- MySQL
- SQL Workbench
- REGEXP
- CAST()
- STR_TO_DATE()
- TRIM()
- REPLACE()
- CASE expressions
